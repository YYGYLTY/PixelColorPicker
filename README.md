## ✨ 项目简介

PixelColorPicker 是一款简单易用的 Android 图片颜色分析工具。

它可以帮助像素艺术创作者、UI 设计师、绘画爱好者以及开发者，
快速分析图片颜色，并转换为可使用的像素色板数据。

主要功能：

- 图片裁剪
- 图片像素化
- 颜色提取
- 游戏色板码生成


## 🎨 功能

### 🖼️ 图片选择

从设备相册选择需要分析的图片。


### ✂️ 自定义裁剪

支持自由调整裁剪区域，
选择需要转换的画面范围。


### 🧩 像素转换

将图片转换为 16×16 像素风格图像。


### 🌈 颜色分析

自动提取像素颜色信息。


### 📋 色板码生成

生成经过压缩编码的游戏可识别色板码。
