# PixelColorPicker

一款简单易用的 Android 图片像素化 / 色板码转换工具。

## ✨ 功能

- 🖼️ 图片选择
- ✂️ 自定义裁剪区域
- 🎨 图片像素化转换
- 🌈 自动提取颜色
- 📋 生成游戏可识别色板码

## 🚀 使用

1. 选择图片
2. 裁剪需要区域
3. 转换为 16×16 像素图
4. 导出颜色数据

## 🛠️ 技术

- Java
- Android SDK
- Bitmap 图像处理
- Protobuf 数据编码
- GZIP 压缩
